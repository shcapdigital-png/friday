PILLAR 01 · CROSS-DOCUMENT
다중 문서 교차 검증
기준파일과 6개 공시서류를 항목별로 비교. 같은 발행조건이 서류 간 다르게 적혀 있으면 즉시 잡아냅니다. 여러 섹션에 흩어진 항목도 모두 추적해 일관성을 확인합니다.
예시 "발행비용율"이 두 곳에 등장 → 두 곳을 모두 찾아 일치 검증
PILLAR 02 · ALGORITHM-BASED
자동 산출 기반 검증
발행일·금리·만기를 자체 계산식으로 다시 산출해, 기준파일 자체의 오기재 가능성까지 차단. 기준이 틀린 채 6종이 일치하는 가장 위험한 시나리오를 막습니다.
예시 "공시일 = 발행일 − 1영업일" 규칙 자동 적용 → 1일 오차 시 사전 경고
PILLAR 03 · STRUCTURED REPORT
표준화 검증 리포트
회차번호 + 타임스탬프로 자동 저장. 페이지·섹션 위치 표시로 내부통제가 요구하는 입증 기록을 그대로 충족합니다.
예시 "불일치 위치: p.34 · 발행제비용 내역" → 검토자가 즉시 이동·추적
